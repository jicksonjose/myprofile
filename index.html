Spice Ecommerce Website

Our E-commerce Website project is a user-friendly platform that caters to two types of users: Admin and Registered Users. The frontend is developed using Bootstrap, ensuring a modern and responsive design that enhances the user experience. The backend is powered by Django and utilizes SQLite for efficient data management. The Admin has the privilege to manage products, update stock levels, and oversee order statuses to ensure smooth operations. On the other hand, Registered Users can easily browse through the product catalog, add items to their shopping cart, and proceed through a seamless checkout process. This website streamlines the online shopping experience, empowering users to make purchases conveniently while allowing the Admin to efficiently manage the entire e-commerce ecosystem.
